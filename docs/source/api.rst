API Endpoints
=============

Data Sources
================
* **USGS Water Services**: Real-time stream gauge data
* **NOAA Tides and Currents**: Tide predictions and observations

Stream Data
-----------

* :code:`https://waikaneappbackend.ees250103.projects.jetstream-cloud.org/api/waikane_stream` -  Get data on Waikāne stream height
* :code:`https://waikaneappbackend.ees250103.projects.jetstream-cloud.org/api/waiahole_stream` - Get data on Waiāhole stream height
* :code:`https://waikaneappbackend.ees250103.projects.jetstream-cloud.org/api/punaluu_stream` - Get data on Punalu'u stream height
* :code:`https://waikaneappbackend.ees250103.projects.jetstream-cloud.org/api/stream_trend` - Get data on stream trends

Tide Data
---------

* :code:`https://waikaneappbackend.ees250103.projects.jetstream-cloud.org/api/waikane_tide_curve` - Gets data on Waikāne’s previous and predicted tide heights
* :code:`https://waikaneappbackend.ees250103.projects.jetstream-cloud.org/api/waikane_tides` - Gets data on Waikāne’s previous and predicted high tides and low tides

Rain data
---------

* :code:`https://waikaneappbackend.ees250103.projects.jetstream-cloud.org/api/rain_data` - Gets data on the amount of rainfall in the last hour and last 6 hours in the Makai and Mauka areas
